# Utils Module / 工具模块

This folder contains utility functions, scripts, and helper classes used throughout the project.
本文件夹包含项目中使用的工具函数、脚本和辅助类。

## Contents / 内容

*   **`gen_pseudo_labels/`**:
    *   Scripts for generating quality pseudo-labels using the SDD method.
    *   使用 SDD 方法生成质量伪标签的脚本。
    *   Includes: `1_gen_datalist.py`, `2_extract_feats.py`, `3_gen_pseudo_labels.py`.

*   **`model/`**:
    *   Model architecture definitions (e.g., ResNet50, IR50).
    *   模型架构定义（如 ResNet50, IR50）。

*   **`train_recognition/`**:
    *   Scripts for training the palmprint recognition backbone.
    *   用于训练掌纹识别主干网络的脚本。

*   **`visualize/`**:
    *   Tools for visualizing results or data.
    *   用于可视化结果或数据的工具。

*   **`dataset_txt.py`**:
    *   Dataset loading utilities for reading image lists and labels.
    *   用于读取图像列表和标签的数据集加载工具。


*   **`image_rename.py`**:
    *   Helper script for batch renaming images.
    *   用于批量重命名图像的辅助脚本。
